How to use NiceNilam

Elements

Options


Background

Color or an image URL for the background.

Use url("Your URL") for image

You can also use linear-gradient() or radial-gradient() to customize your background. (If you are familiar with CSS)

The URL must be collected using this method or provided as a raw image link.

Overlay blur

Blurs the background, set the value to 0px to disable it. Increasing the value will make it blurer.


Border radius

Round the corners of the elements, set the value to 0px to disable it.


Primary text color

Change the text colors for multiple elements all at once, instead of individually. Please note that it will override any other text color settings.


Hover color

Text color when a element is hovered like sidebar items.
